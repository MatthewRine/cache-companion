Import My Finds GPX

  1. Go to geocaching.com → Pocket Queries
  2. Under "My Finds" click "Add to Queue"
  3. Wait for the pocket query to process (you'll get an email)
  4. Download the ZIP file from that same page
  5. Unzip it — you'll find a .gpx file inside
  6. Load that .gpx file here:

The file is parsed entirely in your browser — nothing is uploaded anywhere.
